# To run simulation using vivado tcl script
```
vivado -mode tcl -source run_axil.tcl
```
```
vivado -mode tcl -source run_wb_8.tcl
```

# To run simulation using bash script
## First time initialization
```
chmod +x run_axil.sh
```
```
chmod +x run_wb_8.sh
```

## Using The script
```
./run_axil.sh
```
```
./run_wb_8.sh
```
